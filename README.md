# 🐍 贪吃蛇游戏 - Snake Game

一个使用HTML5 Canvas开发的经典贪吃蛇游戏，具有现代化界面和响应式设计。

## 🎮 游戏特点

- **现代化UI设计** - 渐变背景、发光效果、圆角元素
- **响应式布局** - 适配桌面和移动设备
- **键盘和触摸控制** - 支持键盘(WASD/方向键)和触摸屏
- **分数系统** - 实时计分和最高分记录
- **速度递增** - 蛇越长，移动速度越快
- **本地存储** - 最高分保存在浏览器本地存储中
- **游戏状态显示** - 清晰的状态提示

## 🚀 快速开始

### 在线访问
游戏已部署在GitHub Pages，可以直接访问：
[https://zplswf.github.io/dd/](https://zplswf.github.io/dd/)

### 本地运行
1. 克隆仓库：
   ```bash
   git clone https://github.com/zplswf/dd.git
   ```

2. 进入项目目录：
   ```bash
   cd dd
   ```

3. 使用任何HTTP服务器运行：
   ```bash
   # 使用Python
   python3 -m http.server 8000
   
   # 或使用Node.js的http-server
   npx http-server
   ```

4. 在浏览器中打开：
   ```
   http://localhost:8000
   ```

## 🎯 游戏规则

1. 使用 **方向键** 或 **WASD** 控制蛇的移动方向
2. 吃到红色食物可以增加长度和分数（每个食物+10分）
3. 蛇会随着长度增加而加快移动速度
4. 游戏结束条件：
   - 撞到墙壁
   - 撞到自己的身体
5. 尽可能获得高分！

## 🕹️ 控制方式

### 键盘控制
- **↑ / W** - 向上移动
- **↓ / S** - 向下移动  
- **← / A** - 向左移动
- **→ / D** - 向右移动
- **空格键** - 暂停/继续游戏

### 触摸控制（移动设备）
- 在游戏区域滑动来控制方向

### 按钮控制
- **开始游戏** - 开始新游戏
- **暂停** - 暂停/继续游戏
- **重新开始** - 重置游戏

## 🛠️ 技术栈

- **HTML5** - 页面结构
- **CSS3** - 样式和动画（渐变、阴影、flexbox布局）
- **JavaScript (ES6)** - 游戏逻辑
- **Canvas API** - 游戏图形渲染
- **LocalStorage** - 数据持久化（最高分）

## 🔧 核心功能实现

### 游戏引擎
- 使用Canvas进行2D图形渲染
- 基于网格的移动系统
- 碰撞检测算法
- 游戏循环和状态管理

### 特色功能
1. **蛇身渐变颜色** - 头部到尾部颜色渐变
2. **蛇头眼睛** - 根据移动方向显示眼睛
3. **食物动画** - 发光效果和苹果梗细节
4. **速度系统** - 随长度增加而加速
5. **触摸控制** - 移动设备友好
